# Audio-Tone-Controller
Gemini said Designed as a Baxandall active tone controller in LTspice, this project features a 2nd-order bi-quad filter using OP747 op-amps for high-fidelity audio shaping. A dual-switch RC matrix allows for four distinct response modes—Standard, Treble Boost, Bass Punch, and Full Shaping—enabling precision tuning of bass and treble frequencies.
